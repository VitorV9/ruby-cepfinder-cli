# 🚀 Localizador de CEP

O Localizador de CEP é uma aplicação de linha de comando (CLI) desenvolvida em Ruby com o objetivo de otimizar e automatizar a busca de endereços brasileiros. Consumindo a API pública do ViaCEP de forma totalmente dinâmica, o sistema adota as melhores práticas de Programação Orientada a Objetos para separar a interface do usuário da lógica de rede, além de gerar automaticamente relatórios físicos em texto com o histórico de cada consulta realizada.

---

## 🛠️ Tecnologias

* **Linguagem:** Ruby
* **Dependências:** Bundler, biblioteca json, uri e biblioteca nativa net/http
